How to import the campaign using Google Ads Editor (desktop)

1) Download and install Google Ads Editor
   - Open Google Ads → Tools & Settings → Setup → Google Ads Editor

2) Open Editor and download your account
   - Add account: 507-264-9468
   - Download recent changes

3) Import the CSV
   - In Editor: Account menu → Import → From file…
   - Select: docs/ads/google-ads-editor-import.csv
   - When prompted, ensure “First row contains column names” is checked
   - Review mapping (Editor should auto-detect Types: Campaigns, Ad groups, Keywords, Ads, Negatives)

4) Review campaign settings
   - Campaign name: Mobile Notary — Houston — Search
   - Budget: $10/day
   - Networks: Search only
   - Bidding: Maximize conversions
   - Location targeting: set in the online UI after posting (30-mile radius around 77591)
   - Schedule: 7am–9pm local (set in online UI if needed)

5) Post (upload) the changes
   - Click Post (top right)
   - Confirm upload

6) Finalize in the online Google Ads UI
   - Set Location targeting: Locations → Advanced search → Radius → 30 miles around 77591 → Save
   - Set Ad schedule: 7:00 AM–9:00 PM all days
   - Add call extension: (832) 617-4285 (if not already present)
   - Link GA4 to Ads and import booking_success (if not already):
     • GA4 Admin → Product links → Google Ads → Link (select 507-264-9468)
