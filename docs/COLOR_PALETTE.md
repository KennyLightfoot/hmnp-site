# Color Palette

The project uses a shared color palette defined in `tailwind.config.ts` and `app/globals.css`.

| Token | Hex | HSL | Description |
|-------|-----|-----|-------------|
| `--primary` | `#A52A2A` | `0 59% 41%` | Auburn – primary brand color |
| `--secondary` | `#002147` | `212 100% 14%` | Oxford Blue – secondary brand color |
| `--accent` | `#91A3B0` | `205 16% 63%` | Cadet Gray – accent color |

These values are used in both light and dark themes.
